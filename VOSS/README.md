# RESTCONF API for VOSS

The simple example applications in this directory can demonstrate how a customer would develop an application that communicates with the VOSS REST API.

_restconf.py_ - This is a common module defining the base Restconf Python class that performs:
- basic authentication
- well known URL for the REST interfaces
- get(), post(), patch(), delete() functions
